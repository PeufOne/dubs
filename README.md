## Installation de l'environment de dev

```sh
cp .env backend/.env
cp .env frontend/.env
pnpm install
pnpm run dev -r
```

## Start server

with root user 
```sh
mongod --fork --logpath /var/log/mongodb/mongod.log
```

With dubs user
```sh
cd app
pnpm run --filter ./backend  serve
pnpm run --filter ./frontend  serve
```
